Stanford PwdHash Vault

PwdHash generates theft-resistant passwords. The PwdHash Vault generates these passwords when running in your computer. You can start the Vault at any time by starting the daemon with your master password. If you don't want or can't install the PwdHash Vault, you can generate the passwords using several browser extensions:
